# Data Analysis Projects: Linear Regression Deep Dive

Практический проект по детальному изучению линейной регрессии, включая базовую реализацию, статистическую диагностику и улучшение модели.

## Проекты

### 1. Базовое построение линейной регрессии
Реализация простой и множественной линейной регрессии для прогнозирования непрерывных величин. Изучение фундаментальных концепций алгоритма.

**Что делала:** подготовка данных, визуализация зависимостей, обучение моделей, интерпретация коэффициентов, расчет базовых метрик качества (MSE, MAE, R²).

### 2. Диагностика и проверка статистических предположений
Подробный анализ адекватности линейной регрессионной модели через проверку ключевых статистических предположений.

**Что делала:** анализ остатков модели, проверка линейности связи, тестирование нормальности распределения остатков, оценка гомоскедастичности, проверка на мультиколлинеарность (VIF, корреляционная матрица).

### 3. Обработка проблем данных и улучшение модели
Работа с выбросами, влиятельными наблюдениями и методами улучшения качества регрессионной модели.

**Что делала:** выявление и обработка выбросов, анализ влиятельных точек (расстояние Кука), преобразование переменных, сравнение моделей до и после улучшений.

### 4. Регуляризация: Ridge и Lasso регрессия
Изучение методов регуляризации для борьбы с переобучением и улучшения обобщающей способности модели.

**Что делала:** реализация Ridge и Lasso регрессии, подбор коэффициента регуляризации (alpha), анализ изменения весов признаков при регуляризации, сравнение с обычной линейной регрессией.

## Технологии
Python, Pandas, NumPy, Scikit-learn, Statsmodels, Matplotlib, Seaborn, Jupyter Notebook.
