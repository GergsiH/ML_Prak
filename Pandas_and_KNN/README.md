# Data Analysis Projects: Pandas & KNN Basics

Небольшие проекты для отработки ключевых навыков работы с данными и простыми ML-алгоритмами

## Проекты

### 1. Анализ данных с Pandas
Исследование датасета с использованием Pandas: загрузка, очистка, анализ и визуализация. Основной фокус на освоении основных операций библиотеки для подготовки данных

**Что делала:** обработка пропусков, фильтрация, группировка, построение графиков

### 2. Классификация методом k-ближайших соседей (KNN)
Реализация и применение алгоритма KNN для решения задачи классификации. Проект включает как ручную реализацию логики алгоритма, так и использование готового решения из Scikit-learn для сравнения

**Что делала:** подготовка признаков, подбор параметра k, оценка точности модели, анализ влияния масштабирования данных

## Технологии
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Jupyter Notebook.
