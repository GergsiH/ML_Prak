# Мои проекты по машинному обучению и анализу данных

В этом репозитории собраны мои практические работы по изучению машинного обучения и анализа данных. Каждый проект посвящён отдельному алгоритму или теме, с подробным разбором и реализацией.

## Список проектов

### 1. Линейные модели (Регрессия и Классификация)
Детальное изучение линейных моделей: от базовой линейной регрессии до логистической классификации, включая диагностику и регуляризацию.

### 2. Метод опорных векторов (SVM)
Исследование SVM для классификации и регрессии: работа с линейными и нелинейными данными, подбор ядер и гиперпараметров.

### 3. Деревья решений и ансамбли
Анализ tree-based алгоритмов: построение деревьев решений, изучение Random Forest и других методов ансамблирования.

### 4. Кластеризация и уменьшение размерности
Работа с методами unsupervised learning: K-Means, DBSCAN, PCA и другие техники поиска структуры в данных.

### 5. Нейронные сети и глубокое обучение
Основы нейронных сетей: от простых перцептронов до свёрточных сетей, реализация на Keras/TensorFlow.

### 6. Обработка и анализ данных
Техники предобработки данных, feature engineering и exploratory data analysis (EDA).

## Технологии
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, Jupyter Notebook

## Цель
Систематизировать полученные знания, отработать практические навыки работы с алгоритмами ML и создать структурированное портфолио.
