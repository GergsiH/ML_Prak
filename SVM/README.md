# ML Projects: Support Vector Machines (SVM) Deep Dive

Практический проект, где я подробно изучала метод опорных векторов (SVM) — от базовых принципов до продвинутых техник. Исследовала, как SVM работает для задач классификации и регрессии, экспериментировала с ядрами и параметрами.

## Проекты

### 1. Базовый SVM для линейно разделимых данных
Изучила фундаментальные принципы SVM: как алгоритм строит оптимальную разделяющую гиперплоскость и почему именно опорные вектора играют ключевую роль.

**Что делала:** создавала синтетические линейно разделимые данные, обучала линейный SVM, визуализировала разделяющую гиперплоскость и зазоры (margins), анализировала влияние опорных векторов на результат.

### 2. SVM с различными ядрами для нелинейных данных
Исследовала, как ядерный трюк позволяет SVM работать с данными, которые нельзя разделить линейно. Сравнила разные типы ядер и их параметры.

**Что делала:** работала с нелинейно разделимыми данными (спирали, круги), применяла полиномиальное, RBF и сигмоидное ядра, настраивала степень полинома и параметр gamma, визуализировала получаемые границы решений.

### 3. Подбор гиперпараметров и борьба с переобучением
Углубилась в тонкости настройки SVM: балансировала между точностью на обучении и обобщающей способностью модели.

**Что делала:** исследовала влияние параметров C (штраф за ошибки) и gamma (для RBF ядра) на сложность модели, использовала GridSearchCV и RandomizedSearchCV для оптимального подбора, работала с кросс-валидацией, анализировала learning curves.

### 4. SVM для задач регрессии (SVR)
Применила метод опорных векторов для решения задач регрессии. Изучила, как принцип максимального зазора преобразуется в задачу регрессии.

**Что делала:** реализовала Support Vector Regression (SVR) с разными ядрами, исследовала влияние параметров epsilon (ширина трубки) и C на результат, сравнивала SVR с линейной регрессией на нелинейных данных.

### 5. Работа с реальными данными и масштабированием признаков
Применила SVM к реальным датасетам, уделив особое внимание важности предобработки данных для этого алгоритма.

**Что делала:** стандартизировала и нормализовала признаки, сравнивала производительность SVM до и после масштабирования, работала с текстовыми данными через TF-IDF + SVM, оценивала влияние выбросов на результат.

### 6. Продвинутые техники: кастомные ядра и multiclass классификация
Изучила расширенные возможности SVM, включая создание собственных ядерных функций и работу с задачами, где больше двух классов.

**Что делала:** создавала простые пользовательские ядерные функции, реализовывала стратегии "one-vs-one" и "one-vs-rest" для многоклассовой классификации, сравнивала производительность SVM с другими алгоритмами на сложных датасетах.

## Технологии
Python, Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook, SciPy (для создания кастомных ядер).
